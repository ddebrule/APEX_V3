# Design Spec: Tab 3 - Scribe & Observations
**Aesthetic:** Tactical Field Log

## Visual Layout
- **Upper Section:** "Add Observation" input area (Text + Voice Trigger).
- **Central Section:** "Live Observation Feed" (Chronological bubble/card layout).
- **Sidebar (Optional):** "Detected Signals" - A tag cloud or summary of frequency (e.g., "Bottoming" mentioned 3 times).

## Interactive Elements
- **Voice Trigger:** Large, glowing amber "STATION SCRIBE" button.
- **Signal Highlighting:** Text within notes matches tuning colors (e.g., "Oversteer" in Green text).
- **Edit/Delete:** Subtle context menus on each observation card.

## Colors & Typography
- **Observation Cards:** Subtle glass panels (#1A1A1B).
- **Keywords:** 
    - Handling: #00E676 (Green)
    - Tires: #FF5252 (Red)
    - Signals/Warnings: #FFC107 (Amber)
- **Log Text:** Inter Regular (14px) for readability.
