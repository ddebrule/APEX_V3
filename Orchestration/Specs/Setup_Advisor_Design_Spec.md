# Design Spec: Tab 2 - Setup Advisor
**Aesthetic:** Tactical Diagnostic Console

## Visual Layout
- **Upper Section:** Large "Diagnostic Trigger" buttons (Entry, Apex, Exit, Landing).
- **Central Section:** Split-View Prescription Cards.
    - **Left (Green Border):** "Ideal Performance Fix" (Primary).
    - **Right (Blue Border):** "Quick Trackside Fix" (Alternative).
- **Lower Section:** Implementation History (Mini Audit-Trail).

## Interactive Elements
- **Prescription Cards:** Include "Apply This Change" button.
- **Symptom Selector:** High-contrast buttons with tactile hover states.
- **Logic Visualizer:** Subtle text showing *why* it recommended X (e.g., "Physics Rule: Thicken front diff adds exit on-power steering").

## Colors & Typography
- **Primary Fix:** Highlighted in Electric Green (#00E676).
- **Alternative Fix:** Highlighted in Cyan (#00B0FF).
- **Symptom Active:** High-contrast White text on Charcoal.
